State‑Based Alerts

How to set up alerts (click to expand)
  1. Open the indicator’s Alert Panel (gear icon top‑right).
  2. Select the structural level you want to monitor – e.g., “Daily High”, “NY Gap”, “Holiday Box”.
  3. Choose the condition: Crosses Up, Crosses Down or Touches.
  4. Set the notification method – Discord webhook, email, or in‑app pop‑up.
  5. Save. The alert will fire automatically whenever price interacts with the chosen level.

All alerts are stored in your Coda sheet via the Cloudflare Worker – no extra page required.

Adding the Indicator to Your Chart

One‑click installation (click to expand)
  1. Log into TradingView and open any chart.
  2. Click Indicators → My Scripts → ALL IN 1.
  3. If you haven’t subscribed, you’ll be redirected to the Stripe checkout.
  4. Once the subscription is active, the indicator loads automatically on every new chart.
  5. Use the Settings button to toggle modules (sessions, gaps, ADR, etc.) without leaving the chart.
